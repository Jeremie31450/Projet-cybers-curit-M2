# Projet de Sécurisation du SI d'une PME

## Présentation

Ce projet a été réalisé dans le cadre d'une candidature au titre Expert en Cybersécurité.

L'objectif consiste à concevoir et déployer une stratégie globale de sécurisation du système d'information d'une PME de 150 collaborateurs.

## Technologies utilisées

- Wazuh
- Suricata
- OpenVAS
- Microsoft Entra ID
- Active Directory

## Livrables

- Analyse de risques
- Architecture réseau sécurisée
- Diagramme de Gantt
- Politique IAM
- Rapport d'audit
- Plan de remédiation
